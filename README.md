# CodeAlpha Tasks

This repository contains two tasks for the CodeAlpha assignment:

1. Language Translation Tool (Task 1)
2. FAQ Chatbot (Task 2)

## Files

- `translator.html` → Web-based language translation tool supporting Korean, Chinese, and Japanese.
- `faq_chatbot.html` → Simple FAQ chatbot using JavaScript keyword matching.

## How to Run

1. Download or clone the repository.
2. Open `translator.html` in a web browser to use the translation tool.
3. Open `faq_chatbot.html` in a web browser to use the FAQ chatbot.

## Features

**Translator**
- Supports multiple languages: Korean, Chinese, Japanese
- Uses MyMemory API for translation
- Copy translated text (optional feature)

**FAQ Chatbot**
- Matches user input with stored FAQs
- Provides responses based on keywords
- Displays default message if question is not recognized
